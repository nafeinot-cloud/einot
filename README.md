# einot (תורנות אמא)

אפליקציה למובייל-פירסט (PWA) לניהול ותיאום תורנות אחים (נפתלי, דינה, צורית, מאיר, אבי) לליווי אמא לטיפולים ושהייה איתה בשבתות וחגים.

## תכונות מרכזיות
- **לוח שנה ורשימה:** מעבר מהיר ותצוגה כרונולוגית או חודשית.
- **מחולל שבתות אוטומטי:** יצירת שבתות וחלוקת שיבוץ אוטומטי מחזורי בין האחים (נפתלי ➔ דינה ➔ צורית ➔ מאיר).
- **תאריכים עבריים ופרשת השבוע:** מוצגים אוטומטית ובצורה מקומית (מחושב במכשיר) בכל התצוגות.
- **סנכרון ענן (Firebase):** סנכרון דו-כיווני בזמן אמת לשימוש משותף בין כל האחים.
- **Progressive Web App (PWA):** אפשרות התקנה על מסך הבית בטלפון כאפליקציה עצמאית.
